# BELab 

111-2 BELab Group2 Final Project

- data: 最終的資料集
- data_old: 前面測試、裝置配戴方式未定時的資料集
- imgs: 視覺化 data 資料
- Data_Collection: 蒐集資料用的 Arduino/ Python Code
- Model_Training: CNN Model、SVM Classifier
- Real_Time: 即時演算法的 code
- docs: 三次簡報的投影片和最終 report
- utils: 一些亂七八糟的小程式
